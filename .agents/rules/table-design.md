description: When creating or styling tables
---

1. **Header & Selection**: Our tables should always have the blue header that we use for all our table headers and should be sticky to the top. The font must remain black when not selected, and turn white when the red selection line (`#cd2027` background) appears (without becoming bold).
2. **Font Size**: Ensure that the font size for table headers and body cells is nothing less than `12px` (prefer `13px` for legibility).
3. **Currency Display**: Currency values inside table lists must default to the institution's native currency, **FCFA** (formatted with thousands separator, e.g. `19,500 FCFA`), instead of converting them to other currencies like USD (`$`).
